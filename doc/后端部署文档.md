## 后端部署说明

#### == Python 版本 3.8.3 ==

##### 所需依赖

1. Django 3.0.8
2. Django rest framework 3.11.0
3. PyJWT 1.7.1
4. mysqlclient 2.0.1



## 安装完依赖后序

#### 路径: `mazhi_proj/mazhi_proj/settings.py`


```python
DEBUG = False
ALLOWED_HOSTS = [*]

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': '数据库名',
        'HOST': '数据库主机IP',
        'PORT': '端口号',
        'USER': '用户名',
        'PASSWORD': '密码'
    }
}

```
